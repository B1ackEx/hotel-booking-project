🏨 Hotel Booking System (FastAPI + MongoDB Atlas)

A full-stack hotel booking web application built using **FastAPI**, **HTML/CSS**, and **MongoDB Atlas**.  
Users can submit hotel booking details through a web form, and the data is stored securely in a cloud database.


🚀 Features

- Responsive hotel booking form
- FastAPI backend with REST API
- MongoDB Atlas cloud database integration
- Real-time data storage
- Simple and clean UI
- Deployment ready (Vercel compatible)


🛠 Tech Stack

- Frontend: HTML, CSS
- Backend: FastAPI (Python)
- Database: MongoDB Atlas
- Server: Uvicorn
- Deployment: Vercel
